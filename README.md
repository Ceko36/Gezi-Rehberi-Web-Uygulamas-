# Gezi-Rehberi-Web-Uygulamas-
Proje Adı: Gezi Rehberi Web Uygulaması

Özellikler

MVC Mimarisi
Place.js, Contact.js (veritabanı işlemleri)
EJS şablonları (10+ sayfa)
3 controller (index, places, contacts)
4 route dosyası (index, places, contacts, api)

CRUD İşlemleri

 Create: Yeni gezi yeri ekleme (/places/create)
Read: Yerleri listeleme ve detay görüntüleme
Update: Yer düzenleme (/places/:id/edit)
 Delete: Yer silme (/places/:id/delete)
 
Veritabanı

SQLite3 veritabanı entegrasyonu
2 Tablo: places, contacts
Otomatik tablo oluşturma
Başlangıç verileri (İstanbul, Londra, Paris)

API Endpoints

RESTful API (/api/places, /api/contacts)
JSON formatında veri döndürme

Deploy Hazırlığı

 package.json (bağımlılıklar)
 Procfile (Heroku)
 .gitignore
